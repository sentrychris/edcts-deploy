# EDCTS Deployment Repo

Contains everything required to deploy the EDCTS API and Frontend.

## Server Requirements

### OS / Software

- Ubuntu 22.04.4 LTS (untested on other distributions)
- MySQL 8.0+
- Redis 6.0+
- PHP 8.3 with extensions:
    - Ctype
    - cURL
    - DOM
    - Fileinfo
    - Filter
    - Hash
    - Mbstring
    - OpenSSL
    - PCRE
    - PDO
    - Session
    - Tokenizer
    - XML
    - Zip
    - ZMQ
- Supervisor 4.2+

### Recommended Hardware

- 4 vCPU (2 CPU)
- 16 GB RAM
- 100GB Storage
